Your username without the domain
The domain of your NIP-05 identity
Your Nostr public key in npub format

What is NIP-05?

NIP-05 is a Nostr standard that allows verifying a user's identity by associating an email-like address to a Nostr public key. This facilitates account discovery and verification.

Example

If your NIP-05 identity is alice@example.com, your domain should serve a JSON file at:

https://example.com/.well-known/nostr.json

With content like:

{
  "names": {
    "alice": "b12b632c887f0c871d140d37bcb6e7c1e1a80264d0b7de8255aa1951d9e1ff79"
  }
}

Note: Use hexadecimal format (64 characters), not npub format in the JSON file.
